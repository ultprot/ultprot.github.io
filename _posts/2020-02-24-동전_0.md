# BOJ 동전 0
BOJ 11047 동전 0 문제
[문제](https://www.acmicpc.net/problem/11047)
## C++
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main(void) {
	int n, k;
	cin >> n >> k;
	vector<int> coins(n, 0);
	for (int i = 0; i < n; i++) {
		cin >> coins[i];
	}
	int cursor = n - 1;	//coins의 마지막에 커서 위치
	int answer = 0;
	while (k > 0) {
		answer += k/coins[cursor];
        k %= coins[cursor];
        cursor--;
	}
	cout << answer << endl;
}
```

## Javascript
```js
const readline = require("readline");
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

const lines = [];

rl.on("line", line => {
  lines.push(line)
}).on("close", () => {
  const n = Number(lines[0].split(' ')[0]);
  let k = Number(lines[0].split(' ')[1]);
  
  const coins = lines.slice(1,n+1).map(e=>Number(e));
  
  let answer = 0;
  let cursor = n - 1;
  while(k > 0){
    answer += parseInt(k/coins[cursor]);
    k %= coins[cursor];
    cursor--;
  }
  console.log(answer);
})
```
